**web-aesthetic-master-skill**

This is an Agent Skill designed to significantly improve the aesthetic quality and taste of AI-generated frontends.

Unlike other skills that rely on instructions like “make it look better” or “avoid AI slop,” this skill takes a much more powerful approach: it provides **[85 detailed reference documents for 85 different web design styles](./references)**. Each document includes specific CSS techniques, prohibited patterns, and ready-to-use code snippets.

The AI is required to strictly choose and fully follow one specific style from the library for every project, while completely avoiding generic AI-looking designs, ugly color combinations, cheap gradients, excessive emojis, and other common low-quality patterns.
